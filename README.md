# enuf
The quicker, easier way to **e**lectronically  **nu**mber **f**iles. Because anyone who's ever numbered files by hand has had **enuf** of it. Dedicated to all the bibling lawyers who have done this for generations...

## Installation
From the command line:
```
$ npm i -g enuf 
```

Or, on Windows, you can [download the 64-bit executable by clicking this link](./enuf.exe). Make sure you save it in a folder in your [path](https://www.computerhope.com/issues/ch000549.htm).
